# DDL



```text
show databases;
```

Muestra las bases de datos que existen el motor.

```text
use [nombre de la base de datos]
```

Coloca por defecto una base de datos para poder ejecutar el DDL.

```text
show tables;
```

Muestra las tablas de la base de datos que estamos usando.
